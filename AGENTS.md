# Project Agent Instructions

This repository has no project-specific implementation rules yet.

Agents should follow the user's global instructions and keep any future project
rules in this file only when they are safe for other developers to inherit.

## What Belongs Here

- Build, test, lint, and launch commands for this repository.
- Repository architecture and naming conventions.
- Project-specific coding standards.
- Team-shared workflow rules.

## What Does Not Belong Here

- Personal home-directory paths.
- Local shell aliases or private credentials.
- Machine-specific workflow notes.
- Private briefing or memory locations outside the repository.
